- **Linking**

 If you try to link controllers to any object, controller will jump another place because its parent space is changed. To prevent from this first copy pose, then link controller and paste pose.

 .. image:: /img/tips-tuts/tips/linking.gif
    :width: 400px
	
|
|

- **Duplicating Characters**

 Trying to duplicate character in scene will cause errors. To prevent from this merge same character in to scene as many as you want.

 .. image:: /img/tips-tuts/tips/duplicating.gif
