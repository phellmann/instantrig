Arm & Leg
======================================

If any bone of arm or leg module selected this rollout appears. It allows rigges to create arm or leg modules.

|

Limb Attributes
"""""""""""""""""

.. image:: /img/rigging/createmodule/limbatts.png

Copy: 
 Copy current limb pose.
 
Paste: 
 Paste last copied limb pose.
 
Paste Mirror:
 It pastes copied pose as mirrored.
 
Clavicle: 
 This option adds clavicle bone to limb module.
 
Upper & Lower: 
 It allows rigger to change upper or lower limb twist bones count.

Hand Attributes
"""""""""""""""""

.. image:: /img/rigging/createmodule/handatts.png

If any hand bone selected this rollout appears.

Copy: 
 Copy current hand pose.
 
Paste: 
 Paste last copied hand pose.
 
Finger: 
 Change finger count.
 
Bone: 
 Change bone count for each finger.

Finger Attributes
""""""""""""""""""

.. image:: /img/rigging/createmodule/fingeratts.png

If any finger bone selected this rollout appears.

Copy: 
 Copy current finger pose.
 
Paste: 
 Paste last copied finger pose.
 
Bone: 
 Change bones count for finger.
