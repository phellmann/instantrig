This is the user manual for the |InstantRig| Auto Rigging Tool

|

InstantRig is modular auto rigging tool for 3DS Max users focused keyframe cartoon animations. It helps riggers to create flexible, animator friendly and production quality rigs in minutes.

|

The intention of InstantRig is to help 3d artists, animators, riggers and generalists create great animated works in short time. 

|

|

**InstantRig offers tools below to make things easy and fast without compromising quality:**

- Auto Rigging tool
 
- Auto Skinning tool
 
- Auto Overlapping tool
 
- Copy/Paste/Flip pose manager

|
