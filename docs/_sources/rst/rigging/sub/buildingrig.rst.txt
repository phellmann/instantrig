Building Rig
======================================

If you are done placing bones in scene you are ready to build animation ready rig. Select any part of rig and just press “Build Rig” button in “Main Parameters” rollout. After couple seconds rig will be builded. 

.. warning:: 
   Be sure to save a version before building the rig. It’s not possible to edit rig after building.
